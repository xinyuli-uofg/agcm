# Unlocking the Black Box: Concept-Based Modeling for Interpretable Affective Computing

This repository contains the official code for the paper:

> **Unlocking the Black Box: Concept-Based Modeling for Interpretable Affective Computing Applications**  
> Xinyu Li, Marwa Mahmoud  
> *Proceedings of the IEEE 18th International Conference on Automatic Face and Gesture Recognition (FG 2024)*

The project introduces a **concept-based modeling framework** designed to improve **interpretability** in affective computing systems, helping bridge the gap between model performance and human understanding.
